# Todo-list

## 기능목록

- Todo 생성
    - 제목, 내용, 생성일자, 수정일자
    - 제목 미입력 시 예외발생
    - 생성 시 상태값 : 할일
- Todo 조회
    - 리스트 조회 (제목)
    - 상세 조회 (제목, 내용)
- Todo 수정
    - 제목, 내용 수정
    - 제목 미입력 시 예외발생
- Todo 완료
    - 상태값 : 할일, 완료, 취소
    - 상태값 '취소' 시 리스트에서 제거
- Todo 검색
    - 제목 검색
